--- วิธีการอัปโหลดขึ้น GitHub Pages (ฉบับแก้ไขใหม่) ---

เพื่อป้องกันความสับสนเรื่องชื่อไฟล์ที่ยาวและหายาก ผมได้เปลี่ยนชื่อไฟล์ให้จำง่ายขึ้นแล้วครับ

ไฟล์ที่คุณต้องอัปโหลดขึ้น GitHub (หน้าแรก/Root) มีเพียง 4 ไฟล์นี้เท่านั้น:
1. index.html
2. script.js (คือไฟล์ JS เดิมที่ชื่อยาวๆ)
3. style.css (คือไฟล์ CSS เดิมที่ชื่อยาวๆ)
4. .nojekyll (ถ้าในเครื่องมองไม่เห็น ให้สร้างบน GitHub ตามวิธีที่ผมบอกก่อนหน้าครับ)

ขั้นตอน:
1. ลบไฟล์เก่าบน GitHub ออกให้หมด
2. ดาวน์โหลด ZIP จากโปรเจคนี้ใหม่
3. อัปโหลด 3 ไฟล์ (index.html, script.js, style.css) ขึ้นไป
4. สร้างไฟล์ .nojekyll บน GitHub (ถ้ายังไม่มี)

เพียงเท่านี้หน้าเว็บก็จะทำงานได้ปกติครับ!
