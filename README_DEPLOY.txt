--- IMPORTANT: HOW TO DEPLOY TO GITHUB PAGES ---

The files you see in the root (src, public, package.json, etc.) are SOURCE CODE.
Browsers CANNOT run these files directly.

To fix the "MIME type" error, you must upload the BUILT files:

1. Download the ZIP from Settings -> Export -> Download ZIP.
2. Extract the ZIP.
3. Look for the folder named "public/deploy".
4. UPLOAD ONLY THE CONTENTS of this folder to your GitHub repository.
   (This includes index.html and the assets folder).

DO NOT upload the 'src' folder or 'main.tsx' to the branch you use for GitHub Pages.
